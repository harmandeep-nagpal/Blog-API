# Blog-API
Production-ready Blog API using FastAPI, PostgreSQL, SQLAlchemy, JWT authentication, CRUD APIs, pagination, and search.

## Features Implemented

### Day 1
- FastAPI project setup
- PostgreSQL database configuration
- SQLAlchemy integration
- Database models
- Pydantic schemas
- Automatic table creation using SQLAlchemy ORM

## Tech Stack

- SQLAlchemy

- Python
